西暦・和暦 変換計算機

西暦⇔和暦(明治・大正・昭和・平成・令和)を相互に変換します。履歴書や役所の書類の記入時に便利です。入力した値はお使いの端末内だけで計算され、送信されません。

このツールは年単位の変換です。改元があった年(1912・1926・1989・2019年)は月日によって元号が異なるため、新旧両方の元号を表示します。

使い方

  1. 西暦から調べる場合は上の欄に西暦年を入力して「和暦に変換」を押します。
  2. 和暦から調べる場合は元号と年を選んで「西暦に変換」を押します。

計算方法と根拠

各元号の元年を基準に「西暦 = 元年の西暦 + 和暦年 − 1」で計算しています。

元号と西暦の対応
元号期間(年単位)早見の式
明治1868〜1912年(明治45年まで)西暦 = 1867 + 明治N年
大正1912〜1926年(大正15年まで)西暦 = 1911 + 大正N年
昭和1926〜1989年(昭和64年まで)西暦 = 1925 + 昭和N年
平成1989〜2019年(平成31年まで)西暦 = 1988 + 平成N年
令和2019年〜西暦 = 2018 + 令和N年

出典(一次情報)

  • 元号法(昭和54年法律第43号)
    https://laws.e-gov.go.jp/law/354AC0000000043(2026年7月18日参照)
  • 元号を改める政令(平成31年政令第143号・「令和」の制定)
    https://laws.e-gov.go.jp/law/431CO0000000143(2026年7月18日参照)

計算の前提と注意

  • 改元日は 明治→大正: 1912年7月30日 / 大正→昭和: 1926年12月25日 / 昭和→平成: 1989年1月8日 / 平成→令和: 2019年5月1日 です。改元年の正確な元号は月日でご確認ください。
  • 明治より前の元号(慶応以前)には対応していません。

よくある質問

Q. 令和は西暦何年からですか?

2019年(令和元年)5月1日からです。2019年4月30日までは平成31年でした。

Q. 「昭和100年」は西暦何年にあたりますか?

昭和は64年(1989年)までですが、仮に続いたものとして数える換算では 1925+100=2025年 にあたります。書類の生年月日などには使えない参考値です。

Q. 改元した年の生まれはどちらの元号ですか?

誕生日が改元日より前なら旧元号、以後なら新元号です(例: 1989年1月7日まで昭和64年、1月8日から平成元年)。本ツールは年単位のため両方表示します。
