温度換算 計算機

摂氏(°C)・華氏(°F)・ケルビン(K)を一発で相互変換します。海外レシピのオーブン温度(°F)、アメリカの天気予報、理科・物理の課題に。

使い方

  1. 温度の数値を入力します(小数もOK)。
  2. 入力した数値の単位(°C・°F・K)を選びます。
  3. 「換算する」を押すと、3つの単位すべての値が表示されます。

計算方法と根拠

  • 華氏(°F)= 摂氏(°C)× 1.8 + 32
  • ケルビン(K)= 摂氏(°C)+ 273.15
  • 計算例: オーブンの350°F は(350 − 32)÷ 1.8 = 約177°C
  • 絶対零度(−273.15°C = −459.67°F = 0K)より低い温度は存在しないため、エラーになります。

出典・前提の明記

  • 摂氏とケルビンの関係(t/°C = T/K − 273.15)は国際単位系(SI)の定義によります: 産業技術総合研究所 計量標準総合センター「国際単位系(SI)第9版」
  • 華氏との換算式(°F = °C × 9/5 + 32)は単位の定義に基づく算術計算です。

計算の前提と注意

  • 表示は小数第2位で四捨五入しています。
  • 華氏は主にアメリカで使われる単位です。「華氏100度」は約37.8°Cで、真夏日の気温にあたります。

よくある質問

Q. 華氏350度(オーブン)は摂氏何度ですか?

約177°Cです。海外レシピでよく見る温度は 325°F≒163°C、350°F≒177°C、375°F≒191°C、400°F≒204°C。日本のオーブンなら180°C前後に設定するレシピが多く該当します。

Q. 華氏と摂氏が同じ数値になる温度はありますか?

−40度です。−40°C = −40°F で一致します。これは換算式 F = 1.8C + 32 で F = C とおくと C = −40 になるためです。

Q. ケルビンはいつ使いますか?

物理・化学の計算(気体の状態方程式など)で使う絶対温度の単位です。0K(絶対零度)= −273.15°C が下限で、マイナスの値はありません。目盛りの幅は摂氏と同じです。
